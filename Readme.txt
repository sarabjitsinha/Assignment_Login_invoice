A simple Project on Invoice entry an viewing the pdf file of the Invoice.

User will login and will have access to the home page. 

Dummy data is provided to the user for understanding about the entries.

For convenience purposes, data will persists even after reloads unless the user clears the form field.

Once logged In, Session will remain active unless the user logs out.

Techonlogies used

1> Reactjs
2> Formik 
3> yup
4> Context Api
5> react-pdf
